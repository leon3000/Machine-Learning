    • 介绍
      • 什么是机器学习Machine Learning

        • the field of study that gives computers the ability to learn without being explicitly programmed

          A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E

          • E = the experience of playing many games of checkers
          • T = the task of playing checkers
          • P = the probability that the program will win the next game
      • 监督学习Supervised Learning
        • given a data set and already know what our correct output should look like
        • there is a relationship between the input and the output
        • 分类
          • 回归regression
            • predict results within a continuous output
            • map input variables to some continuous function
          • 分类classification
            • predict results in a discrete output
            • map input variables into discrete categories
      • 无监督学习Unsupervised Learning
        • approach problems with little or no idea what our results should look like
        • can derive structure from data where we don't necessarily know the effect of the variables
        • no feedback based on the prediction results
        • 聚类分析clustering

    • 正则化Regularization
      • 解决过拟合
        • 过拟合overfitting，也叫高方差variance
        • 欠拟合underfitting，也叫高偏差bias
        • 解决过拟合问题的两种方法
          • 减少特征值
            • 手动选择特征值
            • 使用模型选择算法
          • 正则化
            • 保留所有特征值，但是减少θ的幅度
            • 如果特征值高效，正则化作用大
          • 规则化项（也叫权重衰减项），其目的是减小权重的幅度，防止过度拟合
        • 代价函数
          • λ，lambda叫正则化参数
        • 线性回归正则化
            • 通常小于1，假设函数参数θ递减
          • 正规方程Normal Equation
        • 逻辑回归正则化
